# Endpoints

Liste des principaux endpoints de l’API CallShield.

## Santé
- `GET /health` — Vérifie l’état du service.

## Appels
- `POST /call/block` — Bloque un numéro.
- `POST /call/allow` — Autorise un numéro.
- `GET /call/status` — Retourne l’état d’un numéro.

## Utilisateurs
- `GET /user/profile` — Informations du profil.
- `POST /user/update` — Mise à jour des paramètres.
