# Security (API)

Règles de sécurité appliquées à l’API CallShield.

## Principes
- Accès par token Bearer
- Permissions minimales
- Validation stricte des entrées
- Limitation des tentatives d’accès

## Bonnes pratiques
- Rotation régulière des tokens
- Utilisation du HTTPS obligatoire
- Vérification des signatures Webhooks
