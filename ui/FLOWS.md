# Flux utilisateur principaux

## 1. Activation de la protection
Accueil → Activer → Permissions → Protection active

## 2. Consultation d’un appel
Journal → Sélection d’un appel → Détails → Action (bloquer/autoriser)

## 3. Gestion du blocage intelligent
Blocage intelligent → Ajuster niveau → Sauvegarder

## 4. Consultation des alertes
Alertes → Détail → Action rapide

## 5. Configuration initiale
Accueil → Permissions → Paramètres → Protection active
