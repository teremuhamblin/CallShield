# Composants UI de CallShield

## Header
- Titre
- Actions contextuelles

## Card
- Conteneur d’informations
- États : normal, alerte, succès

## ListItem
- Icône
- Titre + sous-texte
- Badge d’état

## Toggle
- Activation/désactivation
- États : on/off

## AlertBanner
- Message court
- Couleur selon criticité

## Button
- Primaire, secondaire
- Icône optionnelle
