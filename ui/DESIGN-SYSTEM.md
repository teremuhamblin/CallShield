# Design System – CallShield v1.0

## Boutons
### Primaire
- Fond : couleur primaire
- Texte : blanc
- États : normal, hover, disabled, loading

### Secondaire
- Fond : transparent
- Bordure : primaire
- Texte : primaire

## Cards
- Padding : M
- Ombre : légère
- États : normal, alerte, succès

## ListItem
- Icône gauche
- Titre + sous-texte
- Badge d’état (autorisé / suspect / bloqué)

## Toggles
- ON : primaire
- OFF : gris
- Animation fluide 150ms

## AlertBanner
- Info : bleu
- Warning : jaune
- Critical : rouge
- Icône + texte court

## Inputs
- Bordure : #D1D5DB
- Focus : primaire
- Label : 14px
