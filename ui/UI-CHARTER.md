# Charte UI – CallShield v1.0

## Couleurs
- Primaire : #0057FF (sécurité, confiance)
- Secondaire : #0A0A0A (sobriété)
- Succès : #0FBF5F
- Alerte : #FFB800
- Danger : #FF3B30
- Fond : #FFFFFF / #F5F7FA

## Typographies
- Titre : Inter Bold, 20–28px
- Sous-titre : Inter Medium, 16–18px
- Corps : Inter Regular, 14–16px
- Légendes : Inter Light, 12px

## Spacing
- XS : 4px
- S : 8px
- M : 16px
- L : 24px
- XL : 32px

## Icônes
- Style : linéaire, minimaliste
- Taille : 20–24px
- Couleur : primaire ou neutre selon contexte

## Composants
- Boutons : arrondi léger (6px), ombre subtile
- Cards : fond blanc, bord fin #E5E7EB
- Listes : séparateurs gris clair
