# Accessibilité – CallShield v1.0

## Contrastes
- Texte normal: ratio ≥ 4.5:1 (AA)
- Texte large: ratio ≥ 3:1 (AA)
- Objectif AAA: 7:1 pour les éléments critiques

## Navigation
- Zones tactiles ≥ 44x44px
- Focus visible sur tous les éléments interactifs
- Ordre logique: haut → bas, gauche → droite

## Typographie
- Taille minimale: 14px
- Interlignage: 1.4
- Éviter les textes en capitales prolongées

## Couleurs
- Ne jamais transmettre une information uniquement par la couleur
- Ajouter icône + texte pour les alertes

## Animations
- Durée max: 200ms
- Aucune animation obligatoire
- Prévoir un mode "réduire les animations"

## Langage
- Messages courts, explicites
- Éviter le jargon technique
- Préférer verbes d’action: “Activer”, “Bloquer”, “Autoriser”
