# Light & Dark Modes – CallShield v1.0

## Mode Light
- Fond: #FFFFFF
- Texte principal: #0A0A0A
- Texte secondaire: #4A4A4A
- Cards: #FFFFFF, bordure #E5E7EB
- Boutons primaires: #0057FF
- Icônes: #0A0A0A

## Mode Dark
- Fond: #0A0A0A
- Texte principal: #FFFFFF
- Texte secondaire: #C7C7C7
- Cards: #1A1A1A, bordure #2A2A2A
- Boutons primaires: #3B82F6
- Icônes: #FFFFFF

## Règles communes
- Contraste minimum: 4.5:1
- Éviter les gris trop proches
- Préserver la hiérarchie visuelle
- Garder les couleurs d’alerte identiques (rouge/jaune/bleu)
